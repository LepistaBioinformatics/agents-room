# Marketing

## Tagline

> **"See your agents. Own your workflow."**

## Descriptive Subtitle

> "A visual workspace for Claude Code agent orchestration. Local. Private. Yours."

## Elevator Pitch

> "Agents Room is a desktop app that gives you a visual canvas to organize, observe, and annotate your Claude Code agents — fully offline, no account required."

---

## Feature Pillars (landing page)

| Pillar | Headline | Description |
|---|---|---|
| Visualize | "Canvas, not terminal" | See all your agents, workspaces, and connections in a zoomable canvas |
| Organize | "Annotate without touching files" | Add notes to agents and workspaces. Your thoughts stay yours |
| Trust | "Offline. No telemetry. No account" | Everything runs on your machine. We don't know you exist |
| Extend | "Skills with trust tiers" | Browse and install skills with clear trust levels |

---

## Screenshots

- **Default to dark mode** for all screenshots and marketing visuals
- Capture the canvas with at least 2 workspace groups visible
- Canvas should show at least one visible connection line
- Do not show personal file paths — use generic workspace names (e.g. "my-project", "council-agents")
- `og:image` dimensions: 1200×630px

## Messaging by Channel

| Channel | Tone | Focus |
|---|---|---|
| GitHub README | Technical, direct | What it does, how to install, how to run |
| GitHub Release | Factual changelog | Features added, bugs fixed, breaking changes |
| Social (X/Twitter) | Brief, developer-focused | One capability per post, dark mode screenshot |
| HN / Reddit | Honest, no hype | Show the problem it solves, invite feedback |
| Lepista ecosystem page | Positioned | Knowledge & AI tool, part of the family |

---

## Anti-patterns (marketing copy)

- No "revolutionary", "game-changing", "AI-powered"
- No feature lists without context — say what the problem is first
- No vague claims — every claim must be verifiable ("fully offline" = no network calls, verifiable in the source)
- Never claim features that don't exist yet
